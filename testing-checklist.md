# Pre-Submission Testing Checklist

## Core Functionality
- [ ] Hover detection works on all link types
- [ ] Preview displays correctly positioned
- [ ] Security scoring shows appropriate values
- [ ] All warning types display correctly
- [ ] Settings save and apply properly

## Performance
- [ ] No lag or slowdown on heavy pages
- [ ] Memory usage stays reasonable
- [ ] Multiple hovers don't cause issues
- [ ] Preview dismisses correctly

## Edge Cases
- [ ] Very long URLs handled properly
- [ ] Special characters in URLs
- [ ] Links in iframes (if possible)
- [ ] Dynamically loaded content
- [ ] Multiple tabs behavior

## Browser Compatibility
- [ ] Works on Chrome latest
- [ ] Works on different screen sizes
- [ ] Dark/light theme compatibility
- [ ] High DPI displays

## Store Requirements
- [ ] Icons in all required sizes
- [ ] Screenshots prepared
- [ ] Privacy policy created
- [ ] Description and metadata ready
- [ ] Permissions justified in description
