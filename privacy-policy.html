Privacy Policy for Link Context Extension

Last updated: December 2024

1. Information We Collect

Link Context does not collect, store, or transmit any personal information.

The extension analyzes URLs you hover over for safety purposes, but:

  • URL analysis happens locally in your browser
  • No personal data is sent to our servers
  • No browsing history is stored or transmitted
  • No user tracking or analytics

2. How We Use Information

The extension only processes URLs temporarily to:

  • Display safety information in the preview tooltip
  • Check link safety using local algorithms
  • Provide security scoring and warnings

3. Data Storage

Link Context stores only:

  • Your extension settings (locally in your browser)
  • Temporary cache of analyzed URLs (cleared automatically)

4. Third-Party Services

Currently, Link Context operates entirely locally without external API calls. If future versions include optional security API integrations:

  • They will be clearly disclosed
  • They will be optional features
  • They will comply with this privacy policy

5. Permissions Explanation

  • activeTab: Required to analyze links on the current webpage
  • storage: Required to save your extension settings
  • host_permissions: Required to work on all websites you visit

6. Your Rights

You have the right to:

  • Use the extension without providing any personal information
  • Disable or uninstall the extension at any time
  • Clear all local data by uninstalling the extension

7. Changes to This Policy

We will notify users of any material changes to this privacy policy through extension updates.

8. Contact

If you have questions about this privacy policy, please contact us through the Chrome Web Store developer contact form.

Link Context is committed to protecting your privacy and providing a safe browsing experience.
